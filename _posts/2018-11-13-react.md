### state 状态
- 只能在组件的constructor中初始化
- 只能 用setState方法更新
- setState会导致render重新执行，渲染组件和子组件

```javascript
constructor(props){
    super(props); //继承父类的props
    this.state={ //初始化
       val:1
    }
}
```
